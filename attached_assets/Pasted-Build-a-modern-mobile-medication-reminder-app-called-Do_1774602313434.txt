Build a modern mobile medication reminder app called “Dose” with 4 screens, bilingual Arabic/English, smart recurring dose alarms, and professional medical-friendly UX
MASTER PROMPT
Create a modern, professional, mobile-first medication reminder app named “Dose”.
The app’s primary purpose is to calculate and display the next dose time for each medication, based on a user-defined recurring interval, and to ring persistent alarms until the user confirms intake.
This app should be designed with excellent UI/UX, feel like a real polished health app, and be optimized for both English and Arabic users, including full RTL support when Arabic is selected.
Use a clean, modern, minimal medical design language with:
Tajawal font for all Arabic text and preferably app-wide typography
Soft rounded corners

Professional medication-management feel
Light and dark themes
Smooth transitions and subtle animations
The app must contain exactly 4 screens:
Splash Screen
Home / Medications Dashboard
Add / Edit Medication Screen
Settings Screen
CORE APP CONCEPT
The app name is Dose.
This app allows the user to:
Add medications
Define medication recurrence intervals
View remaining time until next dose
Receive persistent alarms
Confirm when they actually took the medication
Automatically recycle the next reminder based on the interval
Keep medications in the list even after dose time passes
Delete medications manually only when the user decides they are done
Important behavior:
If a medication is set as every 6 hours, then after the user confirms intake, the app should automatically schedule the next dose exactly 6 hours later, and keep repeating forever until deleted.
If the scheduled time passes and the user does not confirm, the medication must remain in an Overdue state.
The alarm must behave like a mobile alarm, meaning:
It keeps ringing / vibrating / showing notification
It should not stop automatically
It only stops when the user taps “Confirm Intake”
Medications should never disappear automatically
Users can manually delete a medication only by pressing delete
